Minigames!

A set of games made in 1 week each during Lent 2014. Please Enjoy!

Game One Cat Dungeon

2D Platformer. Help the cat find the yarn at the end of the dungeon!

Game Two Harpy's Lair

2D Top Down Boss Battle. The Centaur must defeat the evil Harpy! Use the torches to strengthen your arrows!

Game Three Monster Fighter

3D First Person Menu Style Boss Battle. Can you kill the monster before he kills you?

Game Four Sheep Climb

2D Item Collector. Use the sheep to climb up and collect stars! Watch out though, there are monsters that can eat the sheep! Try for the best time!

Game Five Dragon Raiser

Menu Branching. Raise a dragon to fight off the humans! See if you can raise one of each dragon type!

Game Six Spider

2D Side View Boss Battle. The Wizard Beetle must escape from the Spider! Her webs can freeze you and at half health she will spawn another spider that must be killed!

Game Seven Egg Shooter

3D Shooting Range. Shoot the Easter Eggs!
